# CODTECH SQL Internship Projects

This repository contains all tasks completed during the **CODTECH SQL Internship** using **PostgreSQL**.  
The tasks cover basic to advanced SQL concepts, data migration, and database safety.

---

## Task 1: SQL Joins
**Objective:**  
To combine data meaningfully using different types of joins.

**Concepts Used:**
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN

**Outcome:**  
Learned how to retrieve related data from multiple tables.

---

## Task 2: Advanced SQL Data Analysis
**Objective:**  
To analyze data using complex SQL queries.

**Concepts Used:**
- Common Table Expressions (CTEs)
- Subqueries
- Window Functions

**Outcome:**  
Identified trends, rankings, and patterns from sales data.

---

## Task 3: Database Migration
**Objective:**  
To migrate data from **MySQL to PostgreSQL** while maintaining data integrity.

**Process:**
- Export data from MySQL
- Import data into PostgreSQL
- Verify records using SQL queries

**Outcome:**  
Successfully migrated data without data loss.

---

## Task 4: Database Backup and Recovery
**Objective:**  
To demonstrate database backup and restore in case of failure.

**Tools Used:**
- pg_dump
- pg_restore
- pgAdmin

**Outcome:**  
Learned how to protect data using backups and restore databases when needed.

---

## Tools Used
- PostgreSQL
- pgAdmin
- SQL
- Windows OS

## Conclusion
These tasks helped in understanding SQL from basic queries to real-world database operations such as migration, backup, and recovery.
